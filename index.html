GeoGebra trực tiếp → TikZ / CeTZ

Vẽ hình trực tiếp trong GeoGebra bên dưới, bấm "Sinh code" để lấy ngay code TikZ và CeTZ (Typst) — không cần bước xuất/dán thủ công.
Hướng dẫn dùng & danh sách hỗ trợ (bấm để xem) Cách dùng: Dùng công cụ trên thanh toolbar của GeoGebra để vẽ điểm, đoạn thẳng, đa giác, đường tròn, góc, vector, hàm số, văn bản... như bình thường. Khi vẽ xong, bấm nút "Sinh code" bên dưới — công cụ sẽ tự đọc toàn bộ hình đang có (qua GeoGebra API) và sinh code tương ứng.
Đã hỗ trợ: điểm, đoạn thẳng/tia/đường thẳng, đa giác, đường tròn (tâm+bán kính hoặc tâm+điểm), góc (tự vẽ cung + số đo, tự nhận diện góc vuông), vector có mũi tên, đồ thị hàm số, văn bản tự do, màu sắc, nét đứt, độ dày nét.
Chưa hỗ trợ: đối tượng 3D, danh sách (list), conic phức tạp, hình ảnh chèn vào.

Bấm "Sinh code" hoặc "Xuất PNG" để xuất...
Applet GeoGebra tải từ geogebra.org, preview TikZ dùng TikZJax (tikzjax.com), preview CeTZ dùng typst.ts (jsdelivr) — cả 3 đều cần kết nối mạng và tự tải WASM lần đầu (có thể mất vài giây). Ảnh PNG xuất trực tiếp từ chính GeoGebra (không qua bước dựng lại) nên luôn khớp chính xác hình đang vẽ. Việc sinh code chạy hoàn toàn trong trình duyệt, không gửi hình vẽ đi máy chủ nào khác. Nút "Mở trong Prism" chỉ copy code + mở tab mới do Prism chưa có API công khai để tự động chèn nội dung — Thầy cần dán thủ công vào main.tex.
